UE5 AI Texture & Material Pipeline

Automated Concept-to-Asset Workflow Tool

This Unreal Engine 5 Python tool streamlines the look-development process by bridging the gap between scene blocking and final material application. It leverages a dual-AI integration—Tencent Hunyuan and Meshy AI—to transform simple scene screenshots into production-ready textured assets.

Key Features

  • Intelligent Scene Referencing: Captures automated screenshots of the UE5 viewport to use as a structural base for AI generation.
  • Tencent Hunyuan Integration: Utilizes Hunyuan’s image-style model to generate high-fidelity scene references. Includes a Style Intensity slider, allowing users to balance creative AI interpretation with the original scene’s spatial layout.
  • One-Click Mesh Retexturing: Seamlessly uploads the generated reference image and selected mesh (FBX) to the Meshy AI API.
  • Automated PBR Material Creation: Intelligently handles API callbacks to download generated textures. If PBR mode is enabled, the tool automatically fetches Base Color, Roughness, and Normal maps.
  • Real-time Application: Dynamically creates and assigns new materials to the selected Actor in the scene once the processing is complete.

Technical Stack

  • Engine: Unreal Engine 5 (Python API)
  • Generative AI: Tencent Hunyuan (Image-to-Image / Style Transfer)
  • Texturing AI: Meshy AI (Mesh-to-Texture / PBR Generation)
  • UI: PyQt

The Impact on Art Pipelines

  • Eliminating the “Context Switch”
    Traditionally, an artist must jump between Unreal, a web browser for AI generation, a texturing suite (like Substance), and back to Unreal. This tool keeps the artist inside the viewport, significantly reducing cognitive load and production friction.
  • Rapid Prototyping
    It allows for “instant look-dev.” You can take a greybox level and, within minutes, have a visually cohesive scene that communicates the final art direction to stakeholders or art leads.
  • Controlled AI Integration
    By using the scene’s actual geometry and a screenshot as the foundation, the tool avoids the “randomness” of text-to-image AI. The Intensity setting ensures the AI respects the artist’s original composition and scale.
